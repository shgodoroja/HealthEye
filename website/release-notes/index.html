Track what's changing in HealthEye — new features, workflow improvements, scoring rule updates, and fixes. Score stability is preserved within a major rule version.

1.0 · April 2026

Initial public release

  • Guided Apple Health import on Mac with metric completeness layer
  • Client triage dashboard with attention scoring (recovery, workout, steps, completeness)
  • Explainable score drivers and rule-version transparency
  • Alert rules AR-001 through AR-004 with severity banding
  • One-click weekly PDF report with trend deltas and suggested coach message
  • Data export, archive, and delete controls
  • 14-day trial with Solo Coach and Pro Coach plans
  • macOS and iPadOS support
Known limitations

What is not in 1.0

  • Direct client-to-coach upload portal (use file share today)
  • Team accounts with shared client lists
  • Windows build (macOS and iPadOS only)
  • Automated recurring report scheduling

These items are tracked for future releases. Email hello@healtheye.app to prioritize what matters most for your practice.
