Trust and Explainability

Why coaches can trust the output

Health data is messy. Missing metrics, incomplete exports, and unstable scores create bad coaching decisions. HealthEye is built to make data quality visible before it becomes a problem.

  • Metric completeness is shown per client and per week
  • Missing data is labeled, not silently treated as zero
  • Score breakdown shows what moved the result
  • Score changes are traceable to new data or scoring-rule version updates

You should never have to guess whether a client is flagged because of a real change or a messy import.

Frequently asked questions

Questions coaches and advanced users ask before switching from spreadsheets and ad hoc exports.

Does this work with Apple Health exports from iPhone?+
Yes. HealthEye is built for Apple Health export files from iPhone, including zip exports that contain export.xml. The app guides the import process on Mac.
Do I need to be technical to use it?+
No. The workflow is designed for coaches, not developers. HealthEye guides import, shows data completeness, and explains score changes in plain language.
Is this a medical app?+
No. HealthEye provides wellness and coaching insights only. It is not a medical device and does not provide diagnosis or medical treatment advice.
Can I export or delete client data?+
Yes. Data ownership controls are included so you can export machine-readable data and remove client records when needed.
Does it require an internet connection?+
Core workflows are designed to work locally on Mac, including import, analysis, and report generation.
